Class Function¶

class mipcl_py.mipshell.mipshell.Function(x, points)[source]¶

The Function objects represent piece-wise linear functions y = f(x).

x¶

Var – function argument;

y¶

Var – function value;

points¶

list of pairs (tuples) of float or int – function break points;

hd¶

int – handle that uniquely identifies the function constraint.

__init__(x, points)[source]¶

Initialize a Function object.

Parameters:
  • x (Var) – function argument;
  • points (list of pairs (tuples) of float or int) – function break points.

Note

This constructor initializes just a Function object (not a function constraint). A function constraint is set by the equality operator (see __eq__ below).

__eq__(y)[source]¶

Set a function constrain.

Parameters:y (Var) – variable which value is equal to the value of the function represented by the self object.
Returns:reference to self.
Return type:Function
