Class Term¶

class mipcl_py.mipshell.mipshell.Term(coeff, var)[source]¶

Term objects represent linear terms.

A linear term, a * var, is the product of a constant a and a variable var, which is an object of class Var. In MIPCL-PY applications linear terms are always behind the scene.

coeff¶

float – numeric coefficient;

var¶

Var – reference to a var object.

__eq__(rhs)[source]¶

Process the equality self == rhs.
