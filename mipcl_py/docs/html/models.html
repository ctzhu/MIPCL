The installation bundle of MIPCL-PY contains a substantial number of examples. These example implementations constitute a package named models. Currently this package includes implementations of the following optimization problems.

Problem File
Assembly line balancing assembly.py
Lot sizing with backlogging backloglotsize.py
Short term scheduling in chemical industry batch.py
Bimatrix games bimatrixgame.py
Data envelopment analysis dea.py
Fixed charge network flows fcnf.py
Facility layot fl.py
Generalized assignment genassign.py
Matrix games matrixgame.py
Multidimensional packing multiPacking.py
Multiproduct lot sizing multlotsize.py
Process layout proclayout.py
Project scheduling with limited resources projectScheduling.py
Revenue (yield) management revenue.py
Scheduling problem schedule.py
Transportation problem transport.py
Unit commitment problem (electricity production scheduling) unitcom.py

Many of the models from this table are studied in Mixed Integer Programming in Operations Management.
