Example of usage¶

Fixed charge network flows¶

Let us consider the following MIP

which is a formulation of the fixed charge network flow (FCNF) problem. Here, G=(V,E) is a directed graph, d(v) is demand for some product at node v in V, and, for each e=(v,w) in E, t(e)=v and h(e)=w are the tail and head nodes of arc e, u(e) is the capacity, and f(e)+c(e)x(e) is the the cost of shipping x(e) > 0 units of product along arc e (if x(e)=0, then nothing is paid).

The FCNF problem appears as a subproblem in many practical applications such as design of transportation and telecommunication networks, production planning problems.

An instance of FCNF problem is given in next Figure.

Example of FCNF problem

Here the node numbers and demands are depicted in the upper and lower segments of the node circles. Any (undirected) edge e=(i,j) represents two (directed) arcs e1=(i,j) and e2=(j,i). All horizontal arcs, e, have capacity u(e)=3, fixed cost f(e)=1 and per-unit cost c(e)=0, and all vertical arcs have capacity u(e)=4, fixed cost f(e)=2 and per-unit cost c(e)=0.

  • e[1]: head;
  • e[2]: capacity;
  • e[3]: fixed cost;
  • e[4]: cost.

MIPCL-PY implementation¶

A straightforward MIPshell implementation of the FCNF IP is given in the listing below.

import mipshell
from mipshell import *

def t(e):
    return e[0]
def h(e):
    return e[1]
def u(e):
    return e[2]
def c(e):
    return e[3]
def f(e):
    return e[4]

class Fcnf(Problem):
    def model(self,G):
        self.G = G
        d, E = G['Demands'], G['Edges']
        m, n = len(E), len(d)

        self.x = x = VarVector([m],"x")
        y = VarVector([m],"y",BIN)

        minimize(sum_(f(e)*y[j] + c(e)*x[j] for j,e in enumerate(E)))

        for v in range(n):
            sum_(x[j] for j,e in enumerate(E) if  h(e)==v)\
          - sum_(x[j] for j,e in enumerate(E) if  t(e)==v) == d[v]

        for j,e in enumerate(E):
            x[j] <= u(e)*y[j]

Here, the function model translate the IP model into a MIPCL-PY model.

When a solution has been found, we can print the result using the following procedure, which is a member of class Fcnf.

def printSolution(self):
    if self.is_solution is not None:
        if self.is_solution == True:
            d, E = self.G['Demands'], self.G['Edges']
            x = self.x
            print('Flow cost = {:d}'.format(int(self.getObjVal()+0.5)))
            for j,e in enumerate(E):
                if x[j].val > 0.5:
                    print('flow({:d},{:d}) = {:d}'.format(\
                                  t(e),h(e),int(x[j].val + 0.5)))
        else:
            print('Problem has no solution!')
    else:
        print('Please run optimize first')

Solution of the example FCNF problem¶

#!/usr/bin/python
from fcnf import Fcnf

G = {
  ... # description (see above) of the grid network
}

prob = Fcnf("gridNet")
prob.model(G)
prob.optimize(False)
prob.printSolution()
