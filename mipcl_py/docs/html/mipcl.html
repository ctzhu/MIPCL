Module mipcl¶

Module mipcl is a shared MIPCL-library mipcl.so (mipcl.pyd on windows computers) with an additional interface for calling it in Python programs.

class CLP¶

This class has been designed for solving Linear Programs (LPs).

__init__(self, name)¶
Parameters:name (str) – problem name.
openMatrix(self, m, n, nz, bRowGen, bColGen, mMax = 0, nMax = 0, nzMax = 0)¶

Allocates memory for an LP problem of the required size.

Parameters:
  • m (int) – number of rows;
  • n (int) – number of columns;
  • nz (int) – number of non-zeroes in the matrix;
  • bRowGen (bool) – True if generation of rows is assumed;
  • bColGen (bool) – True if generation of columns is assumed;
  • mMax (int) – maximum number of rows;
  • nMax (int) – maximum number of columns;
  • nzMax (int) – maximum number of non-zeroes in the matrix.
Raises:

CMemoryException – lack of memory.
Note:

If bRowGen=True and mMax=0, openMatrix will estimate mMax; if bColGen=True and nMax=0, openMatrix will estimate nMax.
closeMatrix(self)¶

Does necessary preparations for solving the problem: allocates memory, does preprocessing, scales the matrix.

setVarPriority(self, j, pr)¶

Integer variables can be assigned priorities, i.e., integer numbers from mipshell.PRIO_MIN to mipshell.PRIO_MAX, which are used when selecting a variable for branching. The higher priority the more chances for a fractional variable to be chosen.

Parameters:
  • j (int) – index of variable which priority to be changed;
  • pr (int) – new priority.
setAutoCutPattern(self, nodeNum, height)¶

The solver will automatically generates cuts for first nodeNum processed nodes, and also for nodes of height less or equal than height.

Parameters:
  • nodeNum (int) – number of nodes for which cuts are generated;
  • height (int) – cuts are generated for nodes of height <= height.
